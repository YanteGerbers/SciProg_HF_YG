# SciProg_HF_YH
Hartee Fock program by Yante Gerbers, March 2026

This project is based on a skeleton Hartree Fock repository (SciProg_HF) provided by Lucas Visscher,
for the course Scientific Computing and Programming (University of Amsterdam).

Original code from the repository includes gen1int, interest and interfaces. 
The HarteeFok program in Main.F90 was based on a skeleton HarteeFock program also provided in the original repository.

Modifications and additions (including system setup, SCF implementation, structure adaptations),
were developed by Yante Gerbers.
